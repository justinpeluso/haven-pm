# Phase 4 — Wheels, tires, brakes

## Rear wheel (motor)

| Item | Spec |
|------|------|
| Hub | 36V geared **500W peak** class, OLD per Phase 0/2 (default 135 mm) |
| Rim | 700c, double-wall, disc-compatible, ≥ 36H preferred for hub motors |
| Spokes | Stainless, gauge per lace pattern (often 13G / 2.0 mm on motor side) |
| Lacing | Professional build; hub motors need correct tension & dish |
| Disc | Centerlock or 6-bolt — match hub; **160 mm** rotor |
| Cog | Per Phase 3 (17T SS) |

## Front wheel

| Item | Spec |
|------|------|
| Hub | Disc QR 100×9 or thru 100×12 matching fork |
| Rim | Match rear brand/profile for spare spoke simplicity |
| Spokes | 32H or 36H |
| Rotor | 160 mm |

## Tires & tubes

| Item | Spec |
|------|------|
| Size | **700×35c** primary (range 32–38c) |
| Type | Puncture-resistant road/commuter (e.g. Schwalbe Marathon / Continental Contact class) |
| Tubes | 700×32–40, 48 mm valve if deep rim else 32–40 mm |
| Pressure | Start 60–70 psi rider-dependent; prefer comfort over stock 28c harshness |

Clearance design max: **700×40c**.

## Brakes (required for “perfect”)

Stock Roadster road rim calipers are **out of scope**.

| Item | Spec |
|------|------|
| Type | Hydraulic preferred; mechanical OK (Tektro Aries-class) |
| Rotors | 160 mm front & rear |
| Levers | Match calipers; leave room for display/PAS wires |
| Cutoff | Optional brake-inhibits-motor sensors if controller supports |

Bedding: progressive stops from moderate speed until pads bite consistently.

## Cockpit carryovers (optional)

- Stem 60 mm, bars 550 × 25.4 mm Roadster-like for stealth DNA  
- Or modern 31.8 mm bar + stem if preferred — frame/fork steerer unchanged
