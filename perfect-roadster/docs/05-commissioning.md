# Phase 5 — Integration, tune, legal

## A. Bench electrical (frame optional)

1. Assemble 10S6P pack in trays; verify balance leads mapped B0–B10  
2. Charge to 42.0V; cell-group ΔV ≤ 30 mV after rest  
3. Connect fuse, key, controller, display, PAS, unloaded hub on stand  
4. Confirm PAS levels 1–5 engage smoothly  
5. Clamp DC current under load ≤ controller continuous rating  
6. Verify LVC behavior (controller or BMS)  
7. Insulation: no continuity from either battery polarity to tray screws / mock tube  

**Gate:** Pass A before packing cells into welded frame.

## B. Frame integration

1. Install pack rails, trays, charge port, grommets  
2. Mount controller in bay; dress wires; anti-chafe  
3. Lace/fit rear motor wheel; torque axle; engage torque tab  
4. Install SS drivetrain; set mid-slot tension  
5. Install brakes; center calipers; bed pads  
6. Fit cockpit, display, PAS magnet alignment  
7. Front wheel, tires to pressure  

## C. Ride commissioning

| Test | Pass criteria |
|------|---------------|
| Flat PAS 3 cruise | Stable assist, no cutouts |
| Hill climb PAS 5 | No BMS OD trip; pack wall temp under 50°C after climb |
| Range loop | Log km + Wh if display supports; expect ~2–3× stock Roadster |
| Brake panic stop | Straight, no fade after bedding |
| Chain | No skip; tension OK after 50 km |
| Water | Charge port sealed; no wet connectors |

## D. Legal / class

| Setting | Target |
|---------|--------|
| Assist limit | Class 3: **≤ 28 mph (45 km/h)** where allowed; or local lower limit |
| Throttle | Off / absent unless local rules allow |
| Lights | Required for night street use — wire from display or USB |
| Labeling | Keep personal build notes; know your jurisdiction |

Stock Roadster limited ~24 mph — matching that is a valid stealth choice.
